# Node.js Microservice Backend

This repository contains a collection of microservices built with Node.js and Express.js, including:

- **Average Calculator Microservice**: Calculates the average of streamed number types (`prime`, `fibonacci`, `even`, `random`) using a rolling window.
- **Mock Stock Price API**: Simulates real-time stock price tracking, average calculation over `n` minutes, correlation analysis, and a correlation matrix between stocks.
